# @aura/web

Marketing site for Aura.

**Stack:** Next.js 14 (app router) + Tailwind CSS.

## Sections

1. **Hero** — "Your best friend + life assistant, in one text thread."
2. **How Aura helps** — cards for Relationships, Routines, Headspace.
3. **Text-first experience** — fake iMessage-style chat preview.
4. **Privacy & local-first.**
5. **Who it's for.**
6. **Email capture** — early access.

All copy lives in `lib/copy.ts` so it's easy to tweak.

## Status

🚧 Phase 3 — not yet implemented.
